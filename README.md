**Фриланс-Навигатор** — это комплексная экосистема для специалистов, работающих на себя. Она объединяет в себе личную бухгалтерию, проектный менеджмент и психологию переговоров, позволяя превратить хаотичный фриланс в предсказуемый и прибыльный бизнес.

---

## 🛠 Модули системы

### 💰 1. Экономика жизни и «Месячный минимум» (`hour_calc.html`)
Ваша точка опоры. Преодолейте страх перед ценой, опираясь на реальные цифры.
- **Расчет затрат**: детальный учет личных и деловых расходов.
- **Стоимость простоя**: автоматический учет времени на обучение и поиск клиентов в итоговой ставке.
- **Индикатор нагрузки**: наглядный контроль рабочего графика (защита от переработок).

### 📝 2. Оценщик проектов и Сметы (`project_calc.html`)
Превращаем абстрактное время в профессиональное коммерческое предложение.
- **Умное округление**: алгоритм «бизнес-округления» (до чистого и солидного числа).
- **Разбивка по этапам**: создание прозрачной структуры работ для заказчика.
- **Проверка прибыльности**: автоматическая закладка процента на развитие и налоги.

### ⚖️ 3. Аналитика сделок и Бартера (`index.html`)
Инструмент для оценки неявной выгоды.
- **Коэффициент ликвидности**: стоит ли предлагаемый бартер вашего времени?
- **Анализ партнера**: понимание себестоимости сделки для второй стороны (ключ к усилению вашей позиции).
- **Стратегический вес**: оценка проекта с точки зрения будущего в портфолио.

### 🤝 4. Штурвал переговоров (`negotiation_new.html`)
Ваша психологическая броня при обсуждении цены.
- **Матрица масштаба клиента**: адаптация подхода под разный уровень бизнеса (от локального до федерального).
- **Система «Якорных цен»**: инструменты для управления ожиданиями клиента через методику «Якорь / План / Лимит».
- **12 сценариев сделки**: готовые расчеты для любых условий — от работы за отзывы до борьбы в тендерах.

### 📊 5. Калькулятор амортизации (`amortization_calc.html`)
Профессиональный учет износа оборудования для фотографов, видеографов, диджеев и других специалистов.
- **7 методов расчета**: линейный, уменьшаемого остатка, аннуитетный, единиц производства, групповой, суммы чисел лет, единовременное списание.
- **График амортизации**: наглядная таблица списания стоимости по годам.
- **Простые пояснения**: каждый показатель расшифрован понятными словами с примерами для фрилансеров.

---

## 💎 Особенности
- **Премиальный дизайн**: современный интерфейс с эффектом матового стекла (Glassmorphism), создающий ощущение топового продукта.
- **Живая аналитика**: система сама дает советы профессионального уровня, анализируя ваши вводные данные.
- **Полная автономность**: работает без интернета и серверов. Ваши финансовые секреты остаются только в вашем браузере.
- **Ничего лишнего**: чистый код без тяжелых библиотек и сторонних скриптов.

## 🚦 Как начать путь
Для максимального роста дохода рекомендуется следовать цепочке:
1.  **Посчитайте базу** в разделе расходов — узнайте, ниже какой суммы за час вы не можете опускаться.
2.  **Рассчитайте амортизацию** оборудования — узнайте реальную стоимость съёмки с учётом износа техники.
3.  **Оформите смету** для клиента, используя умное округление — это повышает доверие.
4.  **Изучите сценарий переговоров** — выберите тактику торга и расставьте «ценовые якоря».
5.  **Проверьте бартер**, если клиент предлагает услуги вместо денег — убедитесь в его полезности.
